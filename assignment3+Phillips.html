import numpy as np
import pandas as pd
import os

def freshfruit(f):
    """
    Inputs the file path of the excel file that you are interested in extracting data from
    returns a data frame with the food, form, price/lb, yield, lb/cup and price/cup
    this one gives type as 'fruit' since this is focused on the fruit style
    """
    base = os.path.basename(f)
    base = base[0:base.find('.')]
    doc = pd.read_excel(f,skiprows=1)
    row =doc[doc['Form'].str.startswith('Fresh',na=False)]
    if(row.empty):
        return(None)
    form = str(row['Form'][1])
    priceperlb  = float(row['Average retail price '][1])
    avgyield = float(row['Preparation'][1])
    lbpercup = float(row['Size of a '][1])
    pricepercup = float(row['Average price'][1])
    data = {'type':['Fruit'],'food': [base],'form': [form],'price_per_lb': [priceperlb],'yield': [avgyield],"lb_per_cup":[lbpercup],"price_per_cup":[pricepercup]}
    fruitframe=pd.DataFrame(data =data)
    return(fruitframe)

Price Distribution¶

We wanted to compare the Vegetables against the Fruits for their prices in relation to price per pound. For this we used a boxplot to compare the differences. <img width=400 src="http://puu.sh/tQe31/a5555ce891.png">

From this plot we can see that vegetables tend to be more expensive with a higher median, min, and first and third Quartiles. However, some of the fruits are outliers and tend to be very expensive, these tend to be berries that do not weigh much but are still expensive.

Value Per Price¶

To calculate value per price we wanted to give a penalty for the yield, price per weight should only count for the parts you can actually eat! Look below at the data frame I created to see it sorted by the weighted price per pound which takes the price per pound and divides it by yield. We see that Raspberries are still the most expensive but due to its low yield we see Asparagus fly up the chart. I also compared the weighted price per pound between the fruits and vegetables in a box plot, shown below. <img width=400 src="https://puu.sh/tQf2Z/a3f98016ef.png">
It still shows that vegetables are more expensive on average, but shifts fruit up a little bit and decreases the IQR of Fruit by a noticeable amount.
